\# MultiSig Wallet



\## Description

Multi-signature wallet smart contract implemented in Solidity.



\## Features

\- Multiple owners

\- Required confirmations for transactions

\- Submit / Confirm / Execute / Revoke transactions

\- Protection against duplicate confirmations

\- Only owners can interact

All tests passing (10/10)
